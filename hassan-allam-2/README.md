# Hassan Allam Properties - Landing Page

## Next.js Landing Page - حسن علام للتطوير العقاري

### 🚀 تشغيل المشروع

```bash
# 1. تثبيت الحزم
npm install

# 2. تشغيل في وضع التطوير
npm run dev

# 3. افتح المتصفح على
http://localhost:3000
```

### 🏗️ بناء للإنتاج

```bash
npm run build
npm start
```

### 📁 هيكل المشروع

```
src/
  app/
    globals.css       # الستايلات العامة + فونت Cairo
    layout.js         # الـ layout الأساسي
    page.js           # الصفحة الرئيسية
  components/
    ContactForm.js    # نموذج التواصل
    ProjectSection.js # سيكشن كل مشروع
```
